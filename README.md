# Vagrant setup
  - Install vagrant
  - clone this repo into a new directory
  - run vagrant up

# Files
Following files are included to setup a Vagrant VM for use as a LAMP server.
  - .bash_profile
  - .tmux.conf
  - .vim
  - .vimrc
  - .vimrc-root
  - README.md
  - Vagrantfile
  - lamp-setup.sh
